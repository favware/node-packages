What is Favware?

Favware is the hobbyist organization headed by Jeroen Claassens, a.k.a. Favna. This repository holds all the primary work done under the Favware name in one conventient place. If you have any questions on inquires you can contact Favware at support@favware.tech

Requirements for developing

Before you can develop on packages in this repository you'll have to follow a few steps

  1. Install NodeJS: https://nodejs.org/en/download/
  2. Install Yarn package manager: npm i -g yarn
  3. Globally install CLI tools: yarn global add commitizen eslint jest-cli lerna rollup ts-node typescript
