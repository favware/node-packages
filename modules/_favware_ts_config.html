Module @favware/ts-config

@favware/ts-config

Standardized TSConfig for Favware projects

Install

Install with yarn or npm:

yarn add @favware/ts-config

# npm install @favware/ts-config

Usage

In your tsconfig.json:

{
  "extends": "@favware/ts-config",
  "compilerOptions": {
    "outDir": "dist/",
    "rootDir": "src/"
  },
  "include": ["src"]
}
