## Install

Install with [yarn](https://yarnpkg.com) or [npm](https://www.npmjs.com/):

```sh
yarn add @favware/eslint-config

# npm install @favware/eslint-config
```

## Usage

```json
{
  "extends": ["@favware"]
}
```
